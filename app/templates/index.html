Flask Functions Demo

Welcome to the Flask Functions Demo! This application demonstrates various functions implemented using Flask.

Available Functions:

  • Sleep Function - Sleep for a specified duration.
  • Hello Function - Greet a person with their name.
  • Calculator Function - Perform addition with two numbers.
  • System Info Function - Display information about the system.

Sleep Function

This function accepts a digit as input and sleeps for that many seconds. It then returns the start time and end time of execution.

Hello Function

This function greets a person with their name. Replace John in the URL with any name to see the greeting.

Calculator Function

This function performs addition with two numbers provided as input.

To use this function, submit a POST request to /calculator with JSON data containing num1 and num2.

System Info Function

This function displays information about the system such as the operating system, node name, release, version, and processor.

To use this function, send a GET request to /system_info.
